## Just a script to download audios from [Damma Download](dhammadownload.com) to use in [Mind Peace](https://github.com/PhyoeKyawThan/peace.git)
* Scrap audio link from [https://dhammadownload.com/Chanting-mp3.htm](https://dhammadownload.com/Chanting-mp3.htm)
* Convert downloadable list of direct mp3 url 
* Download each and store under __/audios__
  * collect each audio path in a list with the format I used in [Mind Peace](https://github.com/PhyoeKyawThan/peace.git)
